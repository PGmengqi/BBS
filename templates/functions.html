论坛功能

骨架

已实现:

  • 基于 bootscript 搭建漂亮的主题。
  • 在看不见 navbar 的地方，上滑显示 navbar。动画效果。
  • 时间按秒更新

User

已实现：

  • 数据用 MongoDB 储存。
  • 注册和登陆基本
    • 注册和登陆页面。
    • 字数等基础限制。
    • 密码加密存储。
  • session 保存登陆状态。
  • 任意页面点击登陆/注册，登陆/注册后跳转回之前页面。
    • 从登陆页面进入注册页面，也会跳转到登陆之前页面。
  • 导航栏根据是否登陆动态调整。

BBS

已实现：

  • 数据用 MongoDB 储存。
  • 时间：是以 epoch 开始以秒为单位的float数值的形式保存的。显示的时候在前端用 Javascript 转化为易读的形式。
  • 主页：显示主题的标题、回复数量、点击数量、最近活动。
  • 主题
    • 与用户关联。
    • 用支持 markdown 的编辑器发主题。
    • 使用 Javascript 将帖子内容由 markdown 转换，显示主题内容。
    • 发帖时间，最近活动时间。
    • 点击数。
    • 回复数。
  • 回复
    • 根据是否登陆动态显示回复框或提示登陆的按钮。
    • 发送新回复的请求用 AJAX 实现，并在当前页回复列表的最后，即回复框上方显示新回复。
    • 不管是新回复还是回复列表，都支持 markdown 格式。
    • 用 AJAX 做分页。
    • NEW 载入、换页、新回复，全部用 AJAX 实现。
    • 换页回到页头动画。
  • 面板
    • 基本完成
    • 论坛主页主题可以按面板分类
    • 管理员用户页可以增加面板，同样有动画
    • 帖子页管理员可以将之移动到其他面板,并自动回复以告知
